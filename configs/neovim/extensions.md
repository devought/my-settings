# Neovim Extensions

## Core

- LazyVim - Base distribution

## LSP Servers (via Mason)

- typescript_language_server - TypeScript / JavaScript
- eslint - Linting
- prettier - Formatting

## LazyExtras (via :LazyExtras)

- lang.typescript - TS/JS full support
- lang.json - JSON with schema validation
- lang.docker - Dockerfile support
- lang.tailwind - Tailwind CSS

## Plugins (via Lazy)

- conform.nvim - Format on save
- nvim-lspconfig - LSP configuration
- kdheepak/lazygit.nvim - Git UI
- nvim-treesitter - Syntax highlighting
- blink.cmp - Autocompletion
- neo-tree.nvim - File explorer
- snacks.nvim - File picker / fuzzy finder
- gitsigns.nvim - Git decorations in editor
- lualine.nvim - Status line
- tokyonight.nvim - Colorscheme
- mini.pairs - Auto close brackets
- trouble.nvim - Diagnostics panel
